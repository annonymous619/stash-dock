Quick fixes

Stash says unauthorized

Generate a new key in Stash Settings → Security, paste it into Settings here, save, then run Test Stash connection.

A creator is missing

Confirm the files are under Site / Creator / Album, run Synchronize now, then inspect the sync job log.

A site stopped downloading

Check the job log first. Update the container to refresh gallery-dl and yt-dlp extractors. Some sites require cookies or may block automated access.

Files are in the wrong folder

Check Folder structure and Site labels. Existing files are never moved automatically when a rule changes.
